# seele-todo

一个使用Node.js实现的命令行TODO List工具

### Introduction

#### Installation
SEELE-TODO can be installed as [an npm package](https://www.npmjs.com/package/seele-todo):

`$ npm i seele-todo`

#### Usage

`s [options] [command]`

#### Operations

- `add`: Add a new task. 
- `clear`: Clear all tasks. 
- `show`: Show all tasks and can operate these tasks.
